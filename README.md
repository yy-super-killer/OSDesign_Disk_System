 磁盘文件模拟系统

介绍
华南农业大学SCAU操作系统OS课程设计，JAVAFX开发，可以实现查找文件，磁盘空间使用情况可视化，观察每一个磁盘块的情况。完成文件的基本操作，界面数据实时更新。如果对你有帮助，麻烦点个小星星鸭~
JAVAFX依赖的包放在了lib文件夹中
